$ source schwab-mcp/bin/activate
$ python -m pip install -U pip
$ python -m pip install uv
Install schwab-mcp:
$ uv tool install git+https://github.com/jkoelker/schwab-mcp.git
Install OpenCode to interact with schwab-mcp server:
$ curl -fsSL https://opencode.ai/install | bash
Update PATH in .profile
  if [ -d "$HOME/.local/bin" ] ; then
      PATH="$HOME/.local/bin:$HOME/.opencode/bin:$PATH"
  fi
Get Client ID and Client Secret from https://developer.schwab.com/
Create APP and Select an API Product
  Accounts and Trading Production
  Market Data Production
Enter Callback URL(s)
  https://127.0.0.1:8182/
Wait for status to show - Ready for Use

In VS Code, Install Remote - SSH and Live Preview Extensions.

Authenticate with Schwab to generate a token file.
$ schwab-mcp auth --client-id YOUR_KEY --client-secret YOUR_SECRET --callback-url https://127.0.0.1:8182/
This will open a browser (Click on Open in Browser in VS Code) to log in to Schwab. Once complete, a token will be saved to ~/.local/share/schwab-mcp/token.yaml.
Add MCP Server to OpenCode:
$ opencode mcp add

┌  Add MCP server
│
◇  Enter MCP server name
│  schwab-mcp
│
◇  Select MCP server type
│  Local
│
◆  Enter command to run
│  schwab-mcp server --client-id YOUR_KEY --client-secret YOUR_SECRET
└
Run OpenCode:
$ cd schwab-mcp
$ opencode
Ask OpenCode to:
Show me the options chain for TSLA with Greeks
